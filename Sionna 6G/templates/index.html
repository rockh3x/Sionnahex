📚 Parameter Glossary
BERBit Error Rate — ratio of erroneous bits to total transmitted bits.
BLERBlock Error Rate — ratio of blocks with at least one error.
SNR (Eb/N0)Signal-to-Noise Ratio per bit — the fundamental performance measure.
LDPCLow-Density Parity-Check codes — capacity-approaching FEC used in 5G NR.
MIMOMultiple-Input Multiple-Output — spatial multiplexing using multiple antennas.
OFDMOrthogonal Frequency-Division Multiplexing — multicarrier modulation for 4G/5G/6G.
TDLTapped Delay Line — 3GPP channel models (A-E) for different propagation environments.
EVMError Vector Magnitude — measures modulation quality as a percentage.
ZF / MMSEZero-Forcing / Minimum Mean Square Error equalizers for channel compensation.
Shannon CapacityC = log₂(1 + SNR) — theoretical maximum throughput limit.
